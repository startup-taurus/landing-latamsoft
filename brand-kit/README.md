# LatamSoft Brand Kit

Este paquete contiene una primera identidad visual utilizable para LatamSoft, con versiones para modo claro, modo oscuro, redes sociales y web.

## Archivos principales

- `latamsoft-brand-guide.pdf`: manual de identidad listo para compartir.
- `latamsoft-brand-guide.html`: version editable/visual del manual.
- `latamsoft-brand-guide.md`: version en texto del manual.
- `assets/latamsoft-isologo-light.svg`: isologo horizontal sin fondo para modo claro.
- `assets/latamsoft-isologo-dark.svg`: isologo horizontal sin fondo para modo oscuro.
- `assets/latamsoft-symbol-light.svg`: isotipo/icone LS sin fondo para modo claro.
- `assets/latamsoft-symbol-dark.svg`: isotipo/icone LS sin fondo para modo oscuro.
- `assets/latamsoft-avatar-light.png`: avatar para redes en fondo claro.
- `assets/latamsoft-avatar-dark.png`: avatar para redes en fondo oscuro.
- `assets/latamsoft-social-card-dark.png`: imagen Open Graph para web.
- `assets/latamsoft-linkedin-cover-dark.png`: portada para LinkedIn.
- `web/latamsoft-theme.css`: variables CSS y estilos base para web light/dark.
- `web/brand-tokens.json`: tokens de color, tipografia y radios.
- `reference/latamsoft-original-generated.png`: imagen original aprobada como referencia visual.

## Nota profesional

El logo aprobado fue reconstruido como SVG para que sea escalable y pueda usarse con fondo transparente. Para una identidad final de produccion, conviene vectorizar el wordmark a contornos en una herramienta como Illustrator/Figma cuando se elija la tipografia definitiva.
