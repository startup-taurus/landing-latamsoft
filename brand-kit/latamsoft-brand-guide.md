# LatamSoft - Documento De Identidad Visual

## 1. Esencia De Marca

LatamSoft es una marca de tecnologia creada desde Latinoamerica con vision internacional. Su enfoque combina desarrollo de software a medida, automatizacion de procesos y productos digitales propios.

La idea central de la marca es:

**Software claro, flexible y confiable para simplificar procesos, optimizar recursos y reducir costos.**

La marca debe sentirse joven, visionaria, confiable y creativa. Debe proyectar dominio tecnico sin sonar fria, complicada o demasiado corporativa.

## 2. Propuesta De Valor

**LatamSoft ayuda a empresas a automatizar procesos mediante software a medida y productos digitales, combinando calidad tecnica, innovacion y flexibilidad para simplificar operaciones, optimizar recursos y reducir costos.**

Version corta:

**Software que simplifica procesos, optimiza recursos y reduce costos.**

Version comercial:

**Convertimos procesos complejos en soluciones digitales simples, eficientes y escalables.**

## 3. Publico Objetivo

LatamSoft puede atender distintos tipos de clientes, pero su comunicacion debe hablar principalmente a:

- Empresas y emprendedores que necesitan automatizar procesos.
- Negocios que quieren digitalizar operaciones sin perder flexibilidad.
- Organizaciones que necesitan software confiable, claro y escalable.
- Sectores con necesidades especificas de gestion, como optometria, salud, comercio, servicios profesionales o administracion.

El camino estrategico recomendado es iniciar con servicios a medida y evolucionar hacia productos digitales propios.

## 4. Personalidad

- **Visionaria:** piensa en futuro, escalabilidad y productos.
- **Joven:** agil, moderna y adaptable.
- **Confiable:** ordenada, profesional y segura.
- **Creativa:** propone soluciones, no solo codigo.
- **Clara:** traduce tecnologia compleja en decisiones entendibles.

No debe transmitir: barata, generica, fria, anticuada o excesivamente premium.

## 5. Tono De Comunicacion

El tono recomendado es **tecnico pero claro**.

Usar frases directas, concretas y orientadas a resultados:

- "Automatizamos procesos para que tu equipo trabaje con mas claridad."
- "Creamos software flexible que se adapta a la operacion real de tu negocio."
- "Convertimos ideas y procesos en productos digitales escalables."

Evitar un tono demasiado informal, academico o lleno de jerga tecnica innecesaria.

## 6. Sistema De Logo

La recomendacion profesional es usar un **imagotipo flexible**: simbolo abstracto + wordmark LatamSoft.

En los archivos se entrega tambien como **isologo horizontal** para uso practico de marca. El simbolo LS puede funcionar solo como isotipo para avatar, favicon, app icon y productos digitales.

### Archivos de logo

- `assets/latamsoft-isologo-light.svg` - version principal sin fondo para modo claro.
- `assets/latamsoft-isologo-dark.svg` - version principal sin fondo para modo oscuro.
- `assets/latamsoft-isologo-stacked-light.svg` - version vertical sin fondo para modo claro.
- `assets/latamsoft-isologo-stacked-dark.svg` - version vertical sin fondo para modo oscuro.
- `assets/latamsoft-symbol-light.svg` - simbolo LS sin fondo para modo claro.
- `assets/latamsoft-symbol-dark.svg` - simbolo LS sin fondo para modo oscuro.
- `assets/latamsoft-favicon.svg` - favicon con fondo oscuro.

### Concepto Del Simbolo

El simbolo combina de forma abstracta la **L** y la **S**. La construccion sugiere flujo digital, conexion, automatizacion y movimiento. La referencia a Latinoamerica es sutil: energia, trayectoria y conexion, no mapa literal.

### Uso Principal

- Web corporativa: isologo horizontal.
- Redes sociales: simbolo LS.
- Propuestas comerciales: isologo horizontal o vertical.
- Productos digitales: simbolo LS como icono de producto.
- Favicon: simbolo en contenedor oscuro.

## 8. Tipografia

### Recomendada

- Titulares y marca: **Space Grotesk**
- Texto, UI y documentos: **Inter**
- Alternativa tecnica: **Sora**

### Uso

- Titulares: Space Grotesk Semibold/Bold.
- Parrafos: Inter Regular.
- Botones y etiquetas: Inter Semibold/Bold.
- Evitar tipografias decorativas, serif muy clasicas o fuentes excesivamente futuristas.

## 9. Identidad Para Web

La web debe sentirse clara, tecnologica y orientada a conversion.

### Principios Visuales

- Fondos amplios y limpios.
- Secciones con mucho orden visual.
- Tarjetas sobrias, no decorativas.
- Botones de accion claros.
- Iconografia lineal simple.
- Mensajes orientados a automatizacion, ahorro y claridad.

### Hero Sugerido

Titulo:

**Software que simplifica procesos y acelera operaciones**

Texto:

**Creamos soluciones digitales flexibles para automatizar tareas, optimizar recursos y reducir costos sin complicar tu operacion.**

CTA principal:

**Automatizar mi proceso**

CTA secundario:

**Ver soluciones**

### CSS Incluido

Usar `web/latamsoft-theme.css` como base para modo claro y oscuro. El modo oscuro se activa con:

```html
<body data-theme="dark">
```

## 10. Identidad Para Redes Sociales

### Avatar

Usar siempre el simbolo LS. No usar el logo completo en avatar porque pierde legibilidad en tamaños pequenos.

Archivos recomendados:

- `assets/latamsoft-avatar-dark.png`
- `assets/latamsoft-avatar-light.png`

### Portadas

Usar el logo horizontal, el slogan corto y un fondo oscuro con detalles de conexion. Evitar fondos fotograficos genericos.

Archivo incluido:

- `assets/latamsoft-linkedin-cover-dark.png`

### Posts

Estilo recomendado:

- Fondo oscuro o claro muy limpio.
- Titulares grandes y directos.
- Acentos turquesa/lima solo para enfatizar.
- No saturar con iconos ni mockups innecesarios.

Archivo incluido:

- `assets/latamsoft-post-template-dark.png`

### Pilares De Contenido

- Automatizacion de procesos.
- Desarrollo de software a medida.
- Productos digitales propios.
- Claridad tecnica para negocios.
- Ahorro de tiempo, recursos y costos.

## 11. Usos Correctos

- Mantener siempre la escritura **LatamSoft**.
- Usar suficiente espacio alrededor del logo.
- Usar el simbolo solo en espacios pequenos.
- Usar version dark sobre fondos oscuros.
- Usar version light sobre fondos claros.
- Mantener contraste alto.

## 12. Usos Incorrectos

- No usar mapas literales de Latinoamerica.
- No usar rojo como color de marca.
- No deformar, inclinar o estirar el logo.
- No aplicar sombras pesadas.
- No usar circuitos, engranajes, robots, cohetes o simbolos de codigo como recurso principal.
- No incluir el slogan largo dentro del logo.
- No cambiar la tipografia sin criterio.
- No usar fondos cargados que reduzcan legibilidad.
